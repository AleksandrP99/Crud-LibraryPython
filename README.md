# Books API

RESTful API для управления библиотекой книг. Проект демонстрирует навыки создания API с использованием FastAPI, SQLite и Pydantic.

## 🚀 Технологии

- **FastAPI** - современный веб-фреймворк для Python
- **SQLite** - легковесная реляционная база данных
- **Pydantic** - валидация данных
- **Uvicorn** - ASGI сервер

## 📋 Функционал

- ✅ Создание книги (POST)
- ✅ Получение списка всех книг (GET)
- ✅ Получение книги по ID (GET)
- ✅ Обновление информации о книге (PUT)
- ✅ Удаление книги (DELETE)
